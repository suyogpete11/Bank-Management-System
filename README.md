# Bank Management System

## Objective
To design and implement a C++ based Bank Management Application that simulates core banking operations using object-oriented programming and file handling.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account Details
- File Handling
- OOP Concepts

## Technologies
- C++
- File Handling
- Classes and Objects

## How to Run

Compile:
```bash
g++ bank_management.cpp -o bank
```

Run:
```bash
./bank
